(iOS + Android), and Zalo Mini App. Based in Vietnam, serving users wherever
they interact with AI.

## What we build

| Surface | Stack |
|---|---|
| Web | Next.js (App Router) + TypeScript |
| Mobile | Flutter (Dart); Swift native when needed |
| Zalo | ZMP SDK (situational, VN-market products) |
| Backend | NestJS API + FastAPI AI service |
| Infra | Self-hosted Docker Compose on VPS |
