# The Resonance Team

We are a product engineering organization shipping AI-powered experiences
across web, mobile, and Zalo Mini App. Based in Vietnam.

## What we build

| Surface | Stack |
|---|---|
| Web | Next.js (App Router) + TypeScript |
| Mobile | Flutter (Dart); Swift native when needed |
| Zalo Mini App | ZMP SDK |
| API | NestJS + TypeScript |
| AI service | FastAPI + Python |
| Infra | Self-hosted Docker Compose on VPS |
